Список 2: Інструкція розв’язування квадратного рівняння

  1. Записати коефіцієнти a, b, c у рівнянні ax² + bx + c = 0.
  2. Обчислити дискримінант:
    1. D = b² – 4·a·c
  3. Якщо D ≥ 0, знайти корені:
    1. x₁ = (–b + √D) / (2a)
    2. x₂ = (–b – √D) / (2a)
  4. Якщо D < 0 — коренів у дійсних числах немає.
